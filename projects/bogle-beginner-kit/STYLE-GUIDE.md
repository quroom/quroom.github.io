# Bogle Beginner Kit 작성 가이드 (Bogleheads 철학 기반)

이 폴더의 모든 초보자용 자료는 아래 원칙을 우선한다.

## 핵심 원칙
1. 저비용(비용 최소화)
2. 광범위 분산(시장 전체 접근)
3. 장기 보유(단기 예측 지양)
4. 규칙적 투자(타이밍 예측보다 루틴)
5. 단순한 포트폴리오(관리 가능한 구조)

## 금지/주의
- 단기 급등 종목 추천 금지
- 공포/탐욕 자극형 카피 금지
- 빈번한 매매 유도 금지
- 수익률 보장 표현 금지

## 권장 문장 톤
- "예측보다 원칙"
- "복잡함보다 지속 가능성"
- "완벽한 타이밍보다 꾸준한 실행"

## 품질 체크리스트
- [ ] 초보자가 3분 내 이해 가능한가?
- [ ] 복붙 즉시 실행 가능한가?
- [ ] 장기/저비용/분산 원칙을 훼손하지 않는가?
- [ ] 과장/선동 문구가 없는가?
