## B) 섹션별 적합성 점검 (주 1회)
### 1. 단꿈e 일일분량
- 적합도(1~5): 
- 이유: 
- 다음 주 조정: 분량 유지 / 10% 감소 / 10% 증가

### 2. 영어 루틴(TYM+단어+읽기)
- 적합도(1~5): 
- 어려운 단계: TYM / 단어 / 읽기
- 다음 주 조정: 시간 유지 / 읽기만 축소 / 단어량 축소

### 3. 수학 집코칭(오답 2문제 설명)
- 적합도(1~5): 
- 설명 가능한지: 예 / 아니오
- 다음 주 조정: 2문제 유지 / 1문제로 축소

### 4. 비교 불안 대응
- 적합도(1~5): 
- 효과 있었던 문장: 
- 불안 빈도: 높음 / 중간 / 낮음
