# PRD Draft v1 — Interest-based Bilingual Story English Service

## 0. 문서 상태
- 상태: Draft v1
- 작성 방식: 사용자(학부모)와 공동 설계

## 1. Problem
초5 학습자가 영어를 "공부"로 느끼며 지속성이 떨어지고, 관심 주제 기반 몰입형 읽기/말하기 경험이 부족하다.

## 2. Vision
아이 관심사 기반으로, 한글/쉬운영어 이중 버전 스토리 + 분기형 질문 + 의견 기록을 통해 영어 학습을 게임처럼 지속하게 만든다.

## 3. Target User (초안)
- Primary: 초4~초6, 영어 기초/왕초보~중하위권
- Secondary: 학부모(학습관리자), 교사/튜터(보조 운영)

## 4. Core Value (초안)
1) 관심사 몰입
2) 이해 가능한 영어 입력
3) 클릭 기반 능동 참여
4) 나의 의견 출력(영어/한글)

## 5. MVP Scope (초안)
- 관심 주제 1개 입력
- 챕터 1~3 생성 (한글/쉬운영어)
- 챕터당 분기 질문 3개
- 의견 1~2문장 저장
- 학부모용 주간 리포트(읽은 챕터/응답/단어)

## 6. Open Questions (함께 확정 필요)
1) 첫 MVP 주제 범위 (역사만? 과학/동물 포함?)
2) 영어 난이도 고정(A1~A2) vs 학습자별 조정
3) 이미지 생성 방식 (내장 생성 vs 외부 업로드)
4) 분기 깊이(1단계/2단계)
5) 과금 방식(월 구독/챕터 팩)

## 7. KPI (초안)
- Day7 재방문율
- 세션당 읽은 챕터 수
- 챕터 완주율
- 의견 작성률
- 학부모 만족도(주간)
