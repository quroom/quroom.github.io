# 수능 영어 지문 기반 문항 생성 서비스 계획 v1 (큰 틀)

## 1) 제품 정의
- 입력: 수능/모의고사 스타일 영어 지문
- 출력: 유형별 문제 + 정답 + 해설 + 오답 근거
- 사용자: 학원 강사, 학교 영어교사, N수생 대상 콘텐츠 제작자

## 2) 핵심 가치
1. 지문 1개로 문제 세트 자동 생성(시간 절감)
2. 유형별 포맷 표준화(수업/과제 바로 사용)
3. 해설/근거 자동 제시(검수 시간 단축)

## 3) MVP 범위 (4~6주)
- 문제 유형 3개부터 시작
  - 주제/요지
  - 빈칸 추론
  - 어휘
- 난이도 3단계(하/중/상)
- 출력 형식 2개
  - 학생용(문항)
  - 교사용(정답/해설/근거)
- 교사 검수 모드
  - 생성→수정→저장→PDF/문서 출력

## 4) 기술 구조(하이레벨)
1. Passage Parser
   - 문장 분할/핵심 문장 추출/난이도 지표
2. Item Generator
   - 유형 템플릿 + LLM 생성
3. Validator
   - 정답 유일성, 오답 품질, 근거 매핑 검사
4. Exporter
   - 학생용/교사용 문서 출력

## 5) 품질 기준(KPI)
- 정답 유일성 통과율 95%+
- 교사 수정 없이 채택되는 문항 비율 60%+
- 지문 1개당 생성 시간 60초 이내
- 오답 품질 재작업률 30% 이하

## 6) 저작권/정책 체크 (선행 필수)
- 기존 수능/모의고사 원문 재사용 범위 확인
- 상업 이용 가능 범위(복제/2차 저작물) 법률 검토
- 초기에는 저작권 안전한 자체 지문/허용 데이터셋으로 PoC 진행 권장

## 7) 비슷한 서비스(참고 카테고리)

### A. AI 문제생성/교사용 도구 (해외)
- QuestionWell: https://questionwell.org/
- Diffit: https://www.diffit.me/
- MagicSchool: https://www.magicschool.ai/

### B. 국내 인접 서비스군
- 수능/내신 변형문제 제공 플랫폼(문항뱅크형)
- 학원용 시험지 제작 도구

> 메모: 위 해외 서비스는 “교사용 AI 문항 생성” 레퍼런스로 참고하고,
> 한국 수능 특화는 유형/난이도/해설 포맷에서 차별화 필요.

## 8) 차별화 포인트(권장)
1. 수능 유형 템플릿 고정(국내 실전 적합)
2. 오답 근거를 "왜 틀리기 쉬운지"까지 자동 생성
3. 교사용 검수 워크플로(수정 이력/재생성 버튼)
4. 학습자 레벨별 문항 변형(중하위권/상위권)

## 9) 2주 착수 계획 (디테일 전)
- Week 1
  - 유형 템플릿 3종 정의
  - 평가 루브릭(정답성/난이도/오답품질) 확정
  - 저작권 운영 원칙 문서화
- Week 2
  - 지문 30개로 파일럿 생성
  - 교사 검수 1차 피드백 반영
  - MVP 화면(입력/생성/검수/출력) 와이어 확정

## 10) 오늘 바로 할 일 5개
1. 서비스명 임시 확정
2. MVP 대상 사용자 3명(교사/강사) 인터뷰 질문지 작성
3. 문제 유형 3개 템플릿 초안 작성
4. 저작권 체크리스트 초안 작성
5. 파일럿 지문셋(저작권 안전) 30개 준비
