# 학원 운영 자동화 MVP (OpenClaw)

이 폴더는 **바로 써먹을 수 있는 1차 자동화 묶음**입니다.

## 목표

주말~다음 주부터 바로 체감되는 3가지를 자동화합니다.

1. 문의 응대 초안 자동 생성
2. 결석/과제 리마인드 메시지 자동 생성
3. 주간 운영 리포트 자동 생성

---

## 1) 문의 응대 초안 자동화

### 사용법 (텔레그램에서 나에게 그대로 보내기)

```
[문의응대]
이름: 김OO
학년: 중2
과목: 수학
희망시간: 평일 7시 이후
문의내용: 내신대비반 수업 방식/과제량 궁금
```

### 내가 해주는 것
- 문의 내용을 구조화
- 상담 우선순위(높음/중간/낮음) 분류
- 보호자에게 보낼 **톤 맞는 답장 초안** 생성

---

## 2) 결석/과제 리마인드 자동화

### 사용법

```
[출결리마인드]
대상: 이OO, 박OO
상황: 오늘 무단결석
요청톤: 정중 + 짧게
```

또는

```
[과제리마인드]
대상: 최OO
과목: 영어
미제출: 단어시험 암기표
마감: 내일 18:00
```

### 내가 해주는 것
- 학부모/학생용 문구 분리 생성
- 강한 톤/부드러운 톤 2안 제공
- 필요 시 공지형(단체 발송용)으로 변환

---

## 3) 주간 운영 리포트 자동화

### 사용법

```
[주간리포트]
기간: 2026-02-16 ~ 2026-02-22
데이터:
- 신규 문의 18건
- 등록 5건
- 결석 7건
- 과제 미제출 11건
- 주요 이슈: 중3 시험대비반 문의 증가
```

### 내가 해주는 것
- 한 페이지 요약
- 핵심 지표(전환/리스크/우선 액션)
- 다음 주 실행 항목 3~5개 제안

---

## 운영 팁 (중요)

- 처음 1~2주는 **자동 발송 없이 초안만 생성**
- 개인정보(학생/보호자 실명/연락처)는 최소 입력
- 매일 저녁 1회만 "오늘 운영 요약" 요청하면 누적 관리 쉬움
