# 빠른 명령어 7개

아래 문구를 텔레그램에 붙여서 보내면 됩니다.

1. `문의 1건 정리하고 답장 초안 만들어줘: ...`
2. `오늘 결석자 보호자 안내문 2가지 톤으로 만들어줘: ...`
3. `과제 미제출자 리마인드 문구 만들어줘: ...`
4. `이번 주 운영 리포트 1페이지로 정리해줘: ...`
5. `신규 문의를 등록가능성 높은 순으로 정렬해줘: ...`
6. `휴강 공지 공손하게 작성해줘: ...`
7. `보강 안내 문자 짧게/길게 2안으로 줘: ...`

---

## 추천 운영 루틴
- 매일 마감 전: `오늘 문의/결석/과제 이슈 3줄 요약해줘`
- 일요일 저녁: `주간리포트` 생성
- 월요일 아침: `이번 주 우선 액션 3개` 생성
