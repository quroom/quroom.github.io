# 관심사 기반 이중언어 스토리 영어학습 서비스 계획 v1

## 1) 서비스 한 줄 정의
아이의 관심 주제(예: 2차 세계대전, 대체역사)를 입력하면, **그림 + 한글 버전**과 **그림 + 쉬운 영어 문장 버전**의 스토리북을 만들고, 읽은 뒤 선택형 질문으로 다음 이야기가 이어지는 영어학습 서비스.

---

## 2) 핵심 사용자 가치
1. 아이가 좋아하는 주제로 영어를 하니 학습 지속률이 올라감
2. 같은 내용을 한글/쉬운영어로 병렬 학습해 이해 부담을 낮춤
3. 질문 선택(분기형)으로 수동 독해가 아니라 능동 학습 유도
4. "내 의견 남기기"로 창의적 사고 + 영어 출력(말/글) 훈련

---

## 3) 핵심 기능

### 3-1. 이중 버전 스토리북 생성
- 버전 A: 그림 + 한글 설명
- 버전 B: 같은 장면의 그림 + 쉬운 영어 문장(초5 레벨)
- 단어 난이도 자동 제한(CEFR A1~A2)

### 3-2. 분기형 학습 질문(NotebookLM 스타일 인터랙션)
- 챕터 종료 후 3~5개 질문 카드 제공
- 아이가 클릭하면 다음 장면/이야기 분기
- 질문 유형:
  - 사실 확인(What happened?)
  - 가정 시나리오(What if?)
  - 의견 표현(What do you think?)

### 3-3. 의견 남기기
- 한글/영어 선택 입력
- 문장 프롬프트 제공: "I think... because..."
- 부모/교사용 피드백 코멘트 기능

### 3-4. 학습 지속 장치
- 연속 학습 스트릭
- 관심사 추천 큐(역사/과학/동물/우주 등)
- 다음 이야기 자동 제안

---

## 4) 콘텐츠 예시 (요청 반영)
- 역사: "2차 세계대전"
- 대체역사: "만약 독일이 승리했다면?"

> 주의: 역사/전쟁 주제는 초5에게 맞게 폭력 묘사 최소화, 사실/가상 구분 라벨 명확화 필요.

---

## 5) 학습 설계(초5·왕초보 영어 기준)

### 1세션(20~25분)
1. 한글 버전 5분(맥락 이해)
2. 영어 쉬운문장 버전 10분(소리내어 읽기)
3. 분기 질문 5분(클릭 선택)
4. 의견 1~2문장 5분

### 언어 난이도 규칙
- 한 문장 6~12단어
- 현재형 중심, 복문 최소화
- 새 단어 5개 이하/챕터

---

## 6) MVP 범위 (6주)

### Week 1~2
- 관심사 입력 + 스토리 생성기 v1
- 한글/영어 2버전 동시 출력

### Week 3~4
- 챕터별 분기 질문 엔진
- 클릭형 다음 이야기 이어가기

### Week 5
- 의견 기록 + 히스토리
- 학습 진도/스트릭

### Week 6
- 부모 대시보드(읽은 챕터/의견/단어)
- 파일럿(가정 3~5팀) 테스트

---

## 7) 기술 구조(하이레벨)
1. Topic Engine: 아이 관심사 입력/정제
2. Story Generator: 한글 원본 + 영어 쉬운문장 병렬 생성
3. Branching Engine: 질문 카드 + 분기 노드 생성
4. Notebook/Journal: 의견 저장
5. Parent Dashboard: 진행률/어휘/질문 응답 리포트

---

## 8) 안전/품질 가드레일
- 연령 필터(초5)
- 폭력/혐오/정치 선동성 표현 제한
- 사실/가상 시나리오 라벨링
- 영어 난이도 자동 검사(문장 길이/어휘 레벨)

---

## 9) 비슷한 서비스군 (참고)
- AI 스토리 생성 앱(어린이용 읽기 앱)
- 영어 리딩 앱(레벨 리더 + 퀴즈)
- 인터랙티브 스토리 게임(선택형 분기)

차별화 포인트:
- "관심사 기반" + "한글/쉬운영어 병렬" + "질문 분기" + "의견 기록"의 통합

---

## 10) 오늘 바로 할 일 5개
1. 파일럿 주제 10개 목록 확정(역사/과학/동물/우주)
2. 초5 영어 난이도 규칙표(A1~A2) 확정
3. 분기 질문 템플릿 20개 작성
4. 의견 입력 프롬프트 10개 작성
5. MVP 화면 4개(주제입력/스토리/질문/의견) 와이어 작성
