# 코스피 평균 PER vs 현재 PER, 투기수요 및 12개월 전망 리포트 (V1)

작성일: 2026-02-27

---

## 1) 분석 목적

코스피의 **현재 PER이 장기 평균 대비 어느 위치인지**를 판단하고,
**투기수요(신용융자잔고 등)**를 결합해 향후 12개월 지수 경로를 추정한다.

---

## 2) Executive Summary

- 코스피는 최근 급등 구간으로 **밸류에이션 민감 구간**에 진입.
- 향후 방향은 단순 PER 추가 확장보다 **EPS(이익) 유지/상향 여부**가 핵심.
- 신용융자잔고 증가가 단기 상승 모멘텀을 만들 수 있으나,
  변동성 확대 시 **하방 탄성(급락 리스크)**도 함께 커질 수 있음.
- 현재는 **추세 추종 40 / 리스크 관리 60** 관점이 합리적.

---

## 3) 분석 프레임

### 3-1. 밸류에이션 (현재 PER vs 장기 평균 PER)

판단 로직:

- 현재 PER < 장기평균: 저평가/리레이팅 여지
- 현재 PER ≈ 장기평균: 중립(실적이 방향 결정)
- 현재 PER > 장기평균 + 1σ: 과열 경계

### 3-2. 투기수요(레버리지) 진단 지표

1. 신용융자잔고 절대수준
2. 신용융자잔고 1~3개월 증가율
3. 거래대금/회전율 급증 여부
4. 개인 순매수 편중도(외국인·기관과 괴리)

### 3-3. 가격결정식(실무 단순화)

지수 = EPS × PER

즉, 향후 경로는
- (A) EPS 상향/하향
- (B) PER 확장/압축
조합으로 결정됨.

## 4) 12개월 시나리오 전망

### Bull 시나리오 (확률: 중하)

조건:
- 반도체/AI 중심 EPS 상향 지속
- 외국인 순매수 유지

결과:
- PER 고점 유지 또는 소폭 확장
- 신고점 갱신 가능

리스크:
- 상승 중간 변동성 확대

### Base 시나리오 (확률: 중)

조건:
- EPS 완만 개선
- 금리/환율 중립
- 신용잔고 증가세 둔화

결과:
- PER 정체
- 실적 확인 장세(박스권)

리스크:
- 업종 순환이 매우 빨라져 체감 난이도 상승

### Bear 시나리오 (확률: 중)

조건:
- 신용잔고 과열 + 이익추정 하향 + 대외 충격

결과:
- PER 압축 + 변동성 급등
- 하락 시 낙폭 확대 가능(반대매매 연쇄)

3. **외국인 수급 지속성**: 지수 레벨 유지의 핵심 연료
4. **대형주 쏠림 vs 확산도**: 일부 종목 주도면 피로도 상승 신호
5. **이벤트 변동성**: 실적/정책/환율 쇼크 발생 시 하방 탄성 확대
