# Threads Tools: 전과정 실행 문서 (기획·개발·운영·마케팅)

## 0) 제품 한 줄 정의
"Threads 창작자가 내 게시글을 빠르게 검색하고, 성과를 분석하며, 메모 기반으로 다음 액션까지 연결하는 유료 툴"

---

## 1) 기획 (Week 1~2)

### 문제 정의
- 게시글 누적 후 재활용/찾기 어려움
- 통계와 실제 원문 연결이 불편
- 아이디어 메모와 성과 데이터가 분리됨
- 반응/팔로워 증가에 기여한 패턴을 체계적으로 보기 어려움
- 초안을 벌크로 넣고 일정 간격(예: 1시간) 발행 운영하기 어려움

### 핵심 기능(MVP)
1. 내 글 빠른 검색
2. 성과 대시보드 + 원문 바로가기
3. 스친 메모/태그/검색
4. 미반응 필터(하트/답글 누락 확인)
5. Top 글 탐색 + 패턴 분석(주제/길이/시간대)
6. 벌크 예약 스케줄러(1시간/2시간 간격 배치)

### 성공 지표
- D7 유지율
- 주간 검색 사용 횟수
- 메모 작성률
- Pro 전환율

---

## 2) 개발 (Week 3~6)

### 기술 스택(권장)
- Front: Next.js
- Back: FastAPI or Django
- DB: Postgres (+ pg_trgm/FTS)
- Queue: Redis/Celery or BullMQ

### 모듈
- Auth/OAuth
- Data Ingestion(API/업로드)
- Post Indexer
- Analytics Aggregator
- Notes Service
- Billing(Stripe)

### MVP 범위
- 계정 연결/업로드 중 1개 경로 먼저
- 검색+필터+정렬
- 성과 리스트→원문 이동
- 메모 CRUD + 태그 + 검색
- `미반응 필터`: 내가 아직 반응하지 않은 댓글/멘션 목록
  - 조건: 좋아요 없음 OR 답글 없음
  - 기간: 24시간 / 3일 / 7일
  - 우선순위: 최근순, 관계강도순(자주 상호작용), 영향도순
  - 액션: 하트, 짧은 답글 템플릿, 나중에 보기
- `Top Performance Finder`:
  - 최근 7/30/90일 상위 포스트 자동 정렬
  - 지표: likes/replies/reposts/views/follower_delta
- `Pattern Analyzer`:
  - 주제/글 길이/업로드 시간대별 성과 비교
  - 다음 주 테스트 슬롯 추천
- `Bulk Scheduler`:
  - 초안 N개 일괄 입력
  - 간격(1h/2h/4h), 하루 최대 게시수, quiet hours 설정
  - 정책상 자동 발행 불가 시 반자동(알림+원클릭 복사) 모드

### 테스트
- 데이터 파서 단위 테스트
- 검색 정확도/성능 테스트
- 권한/삭제 흐름 테스트
- 미반응 필터 정확도 테스트(실제 반응 상태와 일치 여부)
- 중복 액션 방지 테스트(같은 대상 반복 하트/답글 방지)
- Top 글 정렬 정확도 테스트(기간/지표별)
- 패턴 분석 검증(시간대/길이 버킷 집계 정확성)
- 벌크 예약 테스트(간격 배치/quiet hours/일일 한도/충돌 처리)

## 3) 운영 (Week 6~8)

### 운영 대시보드
- 가입/활성/결제/해지
- API 실패율
- 동기화 지연시간
- 미반응 누적 건수(24h, 72h)
- 미반응 처리율(하트/답글 완료율)

### 고객 지원
- FAQ
- 데이터 연결 가이드
- 장애 공지 템플릿

### 리스크 관리
- 플랫폼 정책 변경 모니터링
- 수집 경로 fallback(업로드 모드)
- 유료 결제 실패 대응 플로우

---

## 4) 마케팅/세일즈 (Week 6~10)

### 타깃
- Threads에서 주 3회 이상 글 쓰는 창작자/1인사업자

### 런칭 플로우
1. Waitlist 오픈
2. 베타 20명 초대
3. 사례 3개 확보
4. Product Hunt/커뮤니티 공개

### 콘텐츠 전략
- "내 글 200개 중 성과 좋은 글 10초 찾기" 데모
- "통계에서 원문 바로 점프" 사용 영상
- "메모→다음 글 소재 전환" 사례

### 가격
- Free / Pro / Team 3단
- 초기 2개월 얼리버드

## 5) 로드맵

### Phase 1 (MVP, 6주)
- 검색/통계/메모 핵심 기능

### Phase 2 (Growth, 6주)
- AI 요약/재작성 제안
- 게시글 클러스터링

### Phase 3 (Scale)
- 다채널 확장(X, LinkedIn)
- 팀 협업/권한 관리
