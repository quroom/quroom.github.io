# Threads용 보글 유머 카피 100선

포맷: `초보 / 중수 / 고수 / 보글`
톤: 짧은 공감 + 가벼운 풍자 + 원칙 회귀

하락장 알림이 울린 날
- 초보: 알림 끄고도 3분 뒤 다시 켬
- 중수: 새로고침으로 손가락만 단련
- 고수: 리밸런싱 캘린더만 확인
- 보글: 알림보다 원칙 문서 먼저 열기

—-

상승장 헤드라인이 도배된 날
- 초보: 이번엔 내가 천재인가 착각
- 중수: “이번엔 다르다” 세 번째 선언
- 고수: 수익보다 리스크 비중 점검
- 보글: 기분 말고 비중표 확인

—-

친구 수익 인증샷을 본 날
- 초보: 오늘 당장 갈아타기 버튼 탐색
- 중수: 내 전략 긴급회의 소집
- 고수: 비교 대신 내 기준 복기
- 보글: 남의 속도 말고 내 지속성

—-

- 중수: 타이밍 재다가 또 놓침
- 고수: 정해진 날 정해진 금액
- 보글: 의지보다 자동이체 신뢰

- 보글: 공포는 감정 이벤트로 분류
