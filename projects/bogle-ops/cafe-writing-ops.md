# 지수투자 카페 글쓰기 운영 가이드

목표: 매일 1개 이상, 보글 철학 기반 글을 꾸준히 발행한다.

## 톤앤매너 (고정)
- 실전형, 공감형, 과장 금지
- FOMO/지루함/"남의 떡" 심리를 인정하고 다룬다
- 최대수익보다 심리 안정과 지속 가능성을 우선한다
- 기록/자기점검/본업 집중을 행동으로 연결한다
- 마지막은 커뮤니티 참여 질문으로 마무리한다

## 글 템플릿 (권장)
1. 오늘의 문제상황 1문장
2. 왜 흔들리는지 심리 설명 2~3문장
3. 보글 원칙 연결 (저비용/분산/장기/규칙)
4. 실행 체크리스트 2~4개
5. 마무리 질문 + CTA 1문장

## 운영 루틴
- 주제은행(`cafe-topic-bank.md`)에서 `Status: IDEA` 중 1개를 선택
- `Status: DRAFT`로 변경 후 초안 작성
- 발행 완료 시 `PUBLISHED`로 변경 + 발행일 기록
- 발행 후 배운 점 1줄 회고 추가

## 주제 확장 규칙 (매일)
- 기존 주제와 중복되지 않게 최소 3개 신규 추가
- 카테고리 균형 유지: 심리/원칙/실행/세금/포트/생활
- 초보자 관점에서 제목을 질문형 또는 상황형으로 작성
