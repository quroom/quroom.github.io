# 연재 운영 체크리스트 (게시 전/후)

## 1) 게시 전 (5분)
- [ ] 제목이 자극적/예측형 문구를 포함하지 않는가
- [ ] 종목/타이밍/성과경쟁 유도 문구가 없는가
- [ ] 본문이 원칙/구조/기록 중심인가
- [ ] 마무리 질문이 ‘행동유도’가 아닌 ‘기준점검’형인가
- [ ] 운영 메모(댓글 회수 포인트)를 확인했는가

## 2) 게시 후 24시간
- [ ] 댓글 흐름이 원칙 중심인지 확인
- [ ] 추천/예측/성과자랑 댓글은 템플릿으로 정리
- [ ] 공감 댓글 1~2개를 고정해 문화 신호 강화

## 3) 주간 마감
- [ ] 이번 주 반응 좋은 질문 2개 추출
- [ ] 논쟁 유발 문장 2개 기록(다음 글에서 사전 차단)
- [ ] 다음 주 글에 반영할 운영 교훈 1개 정리

## 4) 월간 회고
- [ ] 연재 글별 조회/댓글/저장 지표 정리
- [ ] 문화 적합 댓글 비율 평가
- [ ] 다음 달 연재 톤 미세조정
