# 카페 글 품질 30초 체크리스트

게시 전 30초만 아래 항목 체크하세요.

## 1) 금지선 (하나라도 YES면 수정)
- [ ] 개별 종목/코인 추천 문장이 있다
- [ ] 매수·매도 타이밍 예측 문장이 있다
- [ ] 수익률 자랑/비교를 유도한다
- [ ] FOMO(지금 안 하면 늦는다) 뉘앙스가 있다
- [ ] 타인의 결정을 대신해주는 표현이 있다

## 2) 권장선 (3개 이상 YES 권장)
- [ ] 원칙/구조/습관 중심이다
- [ ] 내 경험/관찰/회고가 포함되어 있다
- [ ] 행동 유도보다 기준 점검 질문이 있다
- [ ] 감정 과열 표현을 줄였다
- [ ] 짧고 명확하게 읽힌다

## 3) 게시판 적합성
- [ ] 자유게시판: 경험/기록/감정/습관 중심
- [ ] 자료게시판: 철학/구조 자료 중심

## 4) 마무리 문장 가이드
- 추천: "정답보다 기준을 확인해봅시다."
- 추천: "맞았는지보다 지켰는지를 점검해봅시다."
- 비추천: "이건 지금 꼭 해야 합니다."

---

## 운영진용 빠른 판정
- PASS: 금지선 0개 + 권장선 3개 이상
- REVIEW: 금지선 0개 + 권장선 2개 이하
- HOLD: 금지선 1개 이상 (수정 요청/정리)
