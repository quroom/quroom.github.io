# Bogle Ops Pack (지수투자 + 카페 운영)

이 문서는 존보글 철학 기반 커뮤니티를 위한 운영 패키지입니다.

구성:
- `investing-system.md` : 지수투자 운영 시스템 (개인 원칙 유지 중심)
- `cafe-ops-manual.md` : 카페 운영 매뉴얼 v1
- `templates.md` : 복붙 가능한 운영/응답 템플릿

핵심 철학:
- 무엇을 살지보다, 무엇을 하지 않을지
- 예측보다 구조
- 수익 자랑보다 원칙 유지
- 설득/논쟁/정답 제시보다 각자 기준 점검

---

## 빠른 실행 순서
1. 카페 공지/필독의 핵심 문구를 `templates.md`의 공지 템플릿으로 통일
2. 신규회원 7일 온보딩 메시지를 예약/반자동 발송 루틴에 넣기
3. 운영진은 `cafe-ops-manual.md`의 정리 기준(삭제/이동/경고)만 동일하게 적용
4. 개인 투자 루틴은 `investing-system.md`의 주간/월간 점검표만 지키기

---

## 30/60/90일 도입 로드맵
- 30일: 규칙 통일 + 온보딩 안정화
- 60일: KPI 추적 + 운영 피로도 감소
- 90일: 코어 문화 정착 + 완만 확장

자세한 계획은 각 문서 하단의 실행 파트 참고.
