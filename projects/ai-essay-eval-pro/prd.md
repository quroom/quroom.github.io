# PRD — AI 서·논술형 평가 시스템

## 목표
- 교사 채점/피드백 업무시간 40% 절감
- 평가 일관성 향상

## MVP 기능
1. 답안 업로드(텍스트/PDF/이미지)
2. 루브릭 템플릿 관리(문법/논리/구성/어휘)
3. AI 1차 점수 + 코멘트 생성
4. 교사 검수/수정 UI
5. 학생/학부모용 리포트 생성

## 제외 범위(MVP)
- 자동 성적 확정
- 학교 행정 시스템 완전 통합

## 사용자 플로우
교사 로그인 → 과제 생성 → 루브릭 선택 → 답안 업로드 → AI 채점 → 교사 수정/승인 → 리포트 발행

## 비기능 요구사항
- 응답속도: 단건 채점 10초 이내(텍스트 기준)
- 감사로그: 채점근거/프롬프트 버전 저장
- 권한: 교사/관리자 분리
