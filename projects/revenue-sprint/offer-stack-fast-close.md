# 즉시수익화 오퍼 3종 재정의 (빠른판매형)

## 원칙
- 1인기업 실행 가능 범위만 판매
- 결과 약속은 "수익"이 아니라 "실행/전환 지표 개선"
- 계약 전 범위/기간/산출물 명확화

## Offer 1) Fast Audit (Starter)
- 기간: 5일
- 가격: 29만~49만원
- 대상: 지금 당장 무엇부터 할지 모르는 1인사업자
- 산출물:
  - 차별점 메시지 1p
  - 오퍼 문장 3종
  - 채널별 CTA 문구
- KPI: 문의/상담 클릭률

## Offer 2) Revenue Sprint (Core)
- 기간: 2주
- 가격: 79만~149만원
- 대상: 콘텐츠는 있는데 전환이 낮은 운영자
- 산출물:
  - 7일 콘텐츠 캘린더
  - DM/댓글 세일즈 스크립트
  - 상담콜 클로징 플로우
- KPI: 상담 신청률, 상담→결제 전환률

## Offer 3) Done-For-You Setup (Premium)
- 기간: 3~4주
- 가격: 199만~399만원
- 대상: 직접 만들 시간 없는 대표
- 산출물:
  - 워크플로우 자동화 세팅
  - 대시보드/운영 SOP
  - 후속 리텐션 루프
- KPI: 반복업무 시간 절감, 응답 SLA 안정화

## 번들/업셀
- Starter -> Core 업셀: 20% 할인
- Core -> Premium 업셀: 진단비 차감

## 클로징 문구 샘플
- "이번 주 안에 실행 가능한 범위만 제안드릴게요."
- "복잡한 설계 대신, 이번 달 매출 흐름에 바로 영향 주는 항목부터 시작합니다."
- "성과는 약속하지 않지만, 실행 구조와 전환 지표는 분명히 개선하겠습니다."

## 고립은둔(예비) 당사자 배려 가이드
- 압박/죄책감 유도 문구 금지
- 선택지를 단계별로 제시(작은 행동 1개부터)
- 연락 빈도/형식에 대한 당사자 동의 우선
- 심리적 안전감을 해치지 않는 표현 사용
