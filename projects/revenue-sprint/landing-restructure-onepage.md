# 랜딩페이지 구조 리팩터링 (1페이지 판매형)

## 목표
- 1페이지에서 "문제 공감 → 해결 제안 → 신뢰 → CTA" 흐름을 완결
- 무료 훅/저가 세션/고가 외주 CTA를 혼선 없이 분리

## 페이지 구조 (복붙형)

### 1) Hero (첫 화면)
- 헤드라인: "이번 주 바로 작동하는 1인사업자 생산성 시스템"
- 서브: "복잡한 컨설팅보다, 실행 가능한 자동화/운영/전환 구조를 만듭니다."
- 버튼 2개:
  - [무료] 10분 진단표 받기
  - [유료] 90분 실전 세팅 신청

### 2) 문제 공감 섹션
- "이런 문제 겪고 있나요?"
  - 반복업무가 많아 핵심 일이 밀림
  - 콘텐츠는 쓰는데 상담/결제로 안 이어짐
  - 문의 응대/후속 루틴이 없음

### 3) 해결 방식(3단계)
1. 무료 진단 (현황 파악)
2. 90분 세팅 (즉시 실행 1개 완성)
3. 2주 스프린트/외주 (구조 고도화)

### 4) 오퍼 카드 3종
- Starter (90분)
- Core (2주 스프린트)
- Premium (1,000~2,000만 외주)

### 5) 신뢰/증거
- 적용 사례 요약 2~3개
- 전/후 지표 1~2개(과장 금지)

### 6) FAQ
- 예산/시간/진행방식/성과보장 여부

### 7) CTA 섹션
- CTA A: "90분 세팅 신청"
- CTA B: "고액 외주 상담 신청"
- 보조 CTA: "무료 진단표 먼저 받기"

## 폼 분리 권장
- 저가 세션 폼: 문제 1개/가능시간/예산 범위
- 외주 상담 폼: 팀규모/목표 KPI/예산/희망 일정

## 카피 가이드
- 금지: 공포 유도, 과장 수익 보장
- 권장: 실행 가능성, 단계형 선택, 현실적 기대치

## 고립은둔(예비) 당사자 배려
- 강한 압박 CTA 금지
- 텍스트 기반 비동기 상담 옵션 표시
- "작게 시작" 문구 명시
