# 2주 MVP 개발 백로그 (수익 우선, 1인기업)

## 목표 (월 100만원 페이스)
- 무료 훅 → 저가 유료 세션 → 2주 구축형 오퍼로 전환되는 최소 퍼널 구축

## 핵심 오퍼
1. 무료: 생산성 자가진단(10분)
2. 저가: 90분 실전 세팅 (OpenClaw/비즈니스메일/랜딩페이지/바이브코딩 중 1개)
3. 메인: 2주 생산성 스프린트

## Week 1 (기획+기본 구축)
### Day 1
- [ ] 오퍼 문장 확정 (누구/무엇/어떻게)
- [ ] 무료 훅(진단표) 초안 작성

### Day 2
- [ ] 랜딩 1페이지(훅/세션/스프린트 CTA)
- [ ] 문의 폼 + 자동응답 텍스트

### Day 3
- [ ] 90분 세션 운영 체크리스트
- [ ] 결제/예약 동선 점검

### Day 4
- [ ] 첫 DM/댓글 응대 스크립트 20개
- [ ] 상담콜 오프닝 스크립트

### Day 5
- [ ] 소프트 오픈(지인/기존 채널 10명)
- [ ] 피드백 수집 및 문구 수정

## Week 2 (검증+개선)
### Day 6~7
- [ ] 무료 훅 배포 콘텐츠 2개
- [ ] 상담 신청 유도 CTA 점검

### Day 8~9
- [ ] 90분 세션 2건 실행
- [ ] 전환 장애요인 기록

### Day 10
- [ ] 랜딩/스크립트 개선 v2
- [ ] FAQ 정리

### Day 11~12
- [ ] 스프린트 제안 2건 발송
- [ ] 거절 사유 분류

### Day 13~14
- [ ] KPI 리뷰(유입/상담/결제)
- [ ] 다음 2주 실행안 고정

## KPI (최소 기준)
- 무료 훅 응답: 30+
- 상담 신청: 6+
- 저가 세션 결제: 3+
- 스프린트 결제: 1+

## 고립은둔(예비) 당사자 배려 원칙
- 강한 압박형 마감 문구 지양
- 단계형 선택지(작은 행동부터) 제공
- 응대 톤: 비판/비교 대신 안전감/명확성
- 필요 시 전문 지원 자원 안내 문구 준비
