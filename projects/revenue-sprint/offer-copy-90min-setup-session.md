# 90분 실전 세팅 세션 판매문구 (복붙용)

## 한 줄 제안
"복잡한 컨설팅 말고, 90분 안에 바로 굴러가는 생산성 세팅 1개를 완성합니다."

## 대상
- 1인사업자/소규모 운영자
- 해야 할 일은 많은데 실행 흐름이 꼬인 상태

## 선택형 세팅 4종 (택1)
1. OpenClaw 기본 자동화 세팅
2. 비즈니스 메일(도메인/서명/템플릿)
3. 랜딩페이지 1p(문의/CTA 연결)
4. 바이브코딩 1:1 입문 실습

## 가격/시간
- 90분 / 9.9만~19.9만

## 결과물
- 당일 적용 가능한 실행 결과 1개
- 다음 7일 운영 체크리스트
- 후속 질문 1회(메시지)

## CTA 문구
- "지금 내 상황에 맞는 90분 세팅, 오늘 바로 예약하기"
- "이번 주 바로 써먹을 세팅 1개부터 시작하기"

## 안내/면책
- 투자수익 보장, 법률/의료 자문은 제공하지 않음
- 고객 상황에 맞춘 실행 설계를 우선

## 고립은둔(예비) 당사자 배려
- 강한 압박 마감 대신 단계형 선택지 제공
- 메시지 톤: 안전감/명확성 중심
