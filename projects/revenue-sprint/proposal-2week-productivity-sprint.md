# 2주 생산성 스프린트 제안서 (복붙용)

## 목적
- 무료 훅 → 저가 세션 → 메인 오퍼 전환 구조를 2주 안에 작동시키는 것

## 기간/비용
- 기간: 2주
- 비용: 79만~149만원 (범위에 따라)

## 제공 범위
1. 오퍼/메시지 정리
2. 랜딩/문의 동선 점검
3. DM/댓글/상담 스크립트 세팅
4. 운영 대시보드 + 주간 실행 루틴

## 산출물
- 핵심 오퍼 문서 1부
- 랜딩 문구/CTA 세트
- 인바운드 세일즈 스크립트
- 주간 KPI 추적표

## KPI
- 문의 수
- 상담 신청률
- 상담→결제 전환률
- 반복업무 시간 절감

## 진행 방식
- 주 2회 점검 + 비동기 피드백
- 실행 중심(문서만 제공하고 끝내지 않음)

## 고객 준비물
- 현재 서비스 소개 자료
- 최근 문의/상담 내역
- 사용 중인 채널/툴 목록

## 고립은둔(예비) 당사자 배려 원칙
- 비난/비교 유도 없는 커뮤니케이션
- 작은 성공경험을 쌓는 단계형 액션
- 필요 시 전문 지원 안내 문구 포함
