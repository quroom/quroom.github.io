# 무료 훅: 1인사업자 생산성 점검 10분 진단표

## 사용 방법
- 각 항목을 0~2점으로 체크
  - 0점: 전혀 안됨
  - 1점: 일부 됨
  - 2점: 안정적으로 됨

## A. 운영 시스템 (최대 10점)
1. 이번 주 우선순위 1개가 명확하다
2. 반복 작업 체크리스트가 있다
3. 문의 응답 SLA(예: 24시간)가 지켜진다
4. 주간 회고/리뷰를 남긴다
5. 작업 상태를 한 곳에서 본다

## B. 마케팅/유입 (최대 10점)
1. 타깃 고객 문장이 명확하다
2. 최근 7일 내 콘텐츠 발행이 있다
3. 콘텐츠마다 CTA가 있다
4. 무료 훅(체크리스트/템플릿)이 있다
5. 채널별 핵심 메시지가 통일되어 있다

## C. 판매/전환 (최대 10점)
1. 90분 유료 세션 같은 첫 결제 상품이 있다
2. DM/댓글 응대 스크립트가 있다
3. 상담콜 오프닝/클로징 흐름이 있다
4. 거절 사유를 기록한다
5. 제안서 전달 후 후속 루틴이 있다

## 점수 해석
- 0~12점: 기본 운영 구조부터 구축 필요
- 13~22점: 유입/전환 연결 개선 필요
- 23~30점: 판매 프로세스 고도화 단계

## 즉시 실행 1개 추천 규칙
- A가 낮으면: 워크큐/체크리스트 먼저
- B가 낮으면: 7일 콘텐츠+CTA 먼저
- C가 낮으면: 90분 세션 상품/스크립트 먼저

## 다음 단계 CTA
- 90분 실전 세팅 세션 신청
- 2주 생산성 스프린트 문의

## 고립은둔(예비) 당사자 배려 문구
- 압박형 표현 대신 선택 가능한 작은 행동 제안
- 개인 속도 존중, 비교/비난 금지
- 필요 시 전문기관/공공지원 연결 안내
