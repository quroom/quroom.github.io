# 월 1회 하이브리드 데모데이 기획

## 1) 목적
- 오프라인 신뢰 형성 + 온라인 확장 동시 달성
- 잠재고객에게 실제 자동화 결과를 체감시키고 상담 전환

## 2) 운영 형태
- 주기: 매월 1회
- 방식: 오프라인(소규모 10~20명) + Zoom 동시 송출
- 시간: 90분

## 3) 프로그램 구성
1. 15분: 오프닝 + 문제 공감
2. 25분: 실제 자동화 데모 2개
3. 20분: 고객 사례(전/후 지표)
4. 20분: 참가자 Q&A
5. 10분: 진단콜/패키지 CTA

## 4) 준비 체크리스트
- [ ] 장소 예약/장비 점검
- [ ] Zoom 링크/신청폼 생성
- [ ] 발표 자료/데모 시나리오 확정
- [ ] FAQ/가격안/신청 링크 준비
- [ ] 촬영/후기 수집 동의 문구 준비

## 5) 전환 흐름
신청 -> 참석 -> 진단콜 신청 -> 제안서 -> 결제

## 6) KPI
- 신청자 수(오프라인/온라인)
- 참석률
- 행사 후 48시간 내 상담 신청률
- 상담->결제 전환율
- 후기/리퍼럴 발생 수

## 7) 운영 팁
- 지역 특성상 오프라인 네트워크 신뢰를 우선
- 데모는 '기능'보다 '시간 절감' 수치 강조
- 행사 후 24시간 내 후속 메시지 발송
