# 자동화 오퍼 1페이지 소개서 (OpenClaw + Make + n8n)

## 한 줄 소개
반복 업무를 자동화해 팀의 실행 속도를 높이는 구축/운영 서비스입니다.

## 이런 분께 맞습니다
- 문의/견적/리포트/알림이 반복되는 1인 사업자
- 자동화를 시도했지만 실제 운영 정착이 안 된 팀
- OpenClaw, Make, n8n 중 무엇이 맞는지 판단이 어려운 경우

## 해결하는 문제
- 반복 업무로 핵심 실행 시간이 줄어듦
- 업무 누락/지연/재작업 발생
- 담당자 의존도가 높아 운영 리스크 증가

## 제공 오퍼
### 1) 자동화 진단 (Entry)
- 현재 업무 흐름 진단
- 자동화 우선순위 맵 제공
- 즉시 실행 가능한 1차 개선안 제시

### 2) 자동화 구축 스프린트 (Core)
- 2주 내 핵심 워크플로우 1~3개 구축
- OpenClaw / Make / n8n 적합 도구 조합
- 테스트 + 운영 인수인계 문서 제공

### 3) 운영 고도화/유지관리 (Recurring)
- 월간 점검 및 개선
- 장애/예외 대응 규칙 정비
- 리포트 자동화 품질 관리

## 기대 효과
- 반복 업무 시간 절감
- 누락/실수 감소
- 운영 표준화 및 재현성 확보

## 작업 방식
1. 문제 정의 (현업 흐름 파악)
2. 설계 (자동화 우선순위 선정)
3. 구축 (핵심 흐름 구현)
4. 정착 (문서화 + 운영 가이드)

## 원칙
- 과장 대신 실행 결과 중심
- 적용 가능/불가능 범위 사전 합의
- 유지 가능한 자동화 우선

## 문의 CTA
반복되는 업무 3가지를 보내주시면,
우선순위와 구축 방향을 바로 제안드리겠습니다.
