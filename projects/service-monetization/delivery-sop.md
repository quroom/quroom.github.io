# 프로젝트 실행 SOP (주간 운영/리포트/장애대응)

## 1) 주간 운영 루틴
### 월요일
- 주간 목표/KPI 확정
- 우선순위 작업 3개 확정

### 수~목요일
- 구축/테스트/중간 데모
- 이슈 로그 업데이트

### 금요일
- 주간 리포트 작성
- 다음 주 개선안 제시

## 2) 주간 리포트 포맷
- 이번 주 완료 항목
- KPI 변화 (시간절감/누락감소)
- 장애/이슈 요약
- 다음 주 계획

## 3) 장애대응 SOP
1. 탐지 (모니터링/고객제보)
2. 분류 (Sev1~3)
3. 공지 (영향도/예상복구)
4. 복구 (핫픽스/롤백)
5. 사후분석 (RCA + 재발방지)

## 4) 변경관리
- 범위 외 요청은 CR로 분리
- 일정/비용 영향도 사전 합의
- 승인 후 반영

## 5) 인수인계 표준
- 운영 문서 전달
- 담당자 교육
- 비상 대응 가이드
- 1주 안정화 지원

## 6) 품질 게이트
- 테스트 통과 확인
- 승인 시연 완료
- 배포 체크리스트 완료
