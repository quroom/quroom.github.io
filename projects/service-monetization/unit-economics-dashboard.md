# 수익성 대시보드 (매출/마진/시간당수익)

## 1) 핵심 지표
- 월 매출 (Revenue)
- 원가/외주비 (COGS)
- 총마진 (Gross Margin)
- 순이익 (Net Profit)
- 시간당 수익 (Hourly Revenue)
- 고객획득비용 (CAC)
- 고객생애가치 (LTV)

## 2) 입력 템플릿
| 항목 | 값 |
|---|---:|
| 월 매출 | 0 |
| 직접 원가 | 0 |
| 마케팅비 | 0 |
| 운영 고정비 | 0 |
| 투입시간(총) | 0 |
| 신규 고객 수 | 0 |
| 구독 고객 수 | 0 |

## 3) 계산식
- Gross Margin = (매출 - 직접원가) / 매출
- Net Profit = 매출 - 직접원가 - 마케팅비 - 고정비
- Hourly Revenue = 매출 / 투입시간
- CAC = 마케팅비 / 신규 고객 수

## 4) 목표 기준선 (초기)
- Gross Margin: 50% 이상
- Hourly Revenue: 목표 시급 기준 이상
- CAC 회수기간: 2개월 이내

## 5) 경고 규칙
- Gross Margin < 40% -> 저수익 경고
- Hourly Revenue 하락 2주 연속 -> 범위/가격 재조정
- CAC 급등 -> 채널 효율 재배분

## 6) 주간 리뷰 질문
1. 가장 수익성 높은 패키지는?
2. 시간 대비 효율이 낮은 작업은?
3. 어떤 채널이 전환에 가장 기여했나?
4. 다음 주 가격/오퍼 실험은 무엇인가?
