# 수익성 대시보드 (매출/마진/시간당수익)

## 1) 핵심 지표
- 월 매출 (Revenue)
- 원가/외주비 (COGS)
- 총마진 (Gross Margin)
- 순이익 (Net Profit)
- 시간당 수익 (Hourly Revenue)
- 고객획득비용 (CAC)
- 고객생애가치 (LTV)

## 2) 입력 템플릿
| 항목 | 값 |
|---|---:|
| 월 매출 | 0 |
| 직접 원가 | 0 |
| 마케팅비 | 0 |
| 운영 고정비 | 0 |
| 투입시간(총) | 0 |
| 신규 고객 수 | 0 |
| 구독 고객 수 | 0 |

## 3) 계산식
- Gross Margin = (매출 - 직접원가) / 매출
- Net Profit = 매출 - 직접원가 - 마케팅비 - 고정비
- Hourly Revenue = 매출 / 투입시간
- CAC = 마케팅비 / 신규 고객 수

## 4) 목표 기준선 (초기)
- Gross Margin: 50% 이상
- Hourly Revenue: 목표 시급 기준 이상
- CAC 회수기간: 2개월 이내

## 5) 경고 규칙
- Gross Margin < 40% -> 저수익 경고
- Hourly Revenue 하락 2주 연속 -> 범위/가격 재조정
- CAC 급등 -> 채널 효율 재배분

## 6) 주간 리뷰 질문
1. 가장 수익성 높은 패키지는?
2. 시간 대비 효율이 낮은 작업은?
3. 어떤 채널이 전환에 가장 기여했나?
4. 다음 주 가격/오퍼 실험은 무엇인가?


## 7) 리드 우선순위 운영 지표 (Top5 Rule)
| 항목 | 값 |
|---|---:|
| 오늘 Top5 후보 수 | 5 |
| Top5 실제 접촉 수 | 0 |
| Top5 응답 수 | 0 |
| Top5 상담 전환 수 | 0 |
| Top5 제안서 발송 수 | 0 |

- Top5 응답률 = Top5 응답 수 / Top5 실제 접촉 수
- Top5 상담전환률 = Top5 상담 전환 수 / Top5 응답 수


## 8) 채널/퍼널/매출 확장 지표
| 지표 | 정의 | 목표 |
|---|---|---:|
| 채널별 리드 수 | LinkedIn/Threads/지역별 주간 리드 | 주간 20+ |
| 리드→상담 전환율 | 상담 수 / 리드 수 | 20%+ |
| 상담→제안 전환율 | 제안 수 / 상담 수 | 60%+ |
| 제안→계약 전환율 | 계약 수 / 제안 수 | 25%+ |
| 평균 계약금액 | 계약금액 평균 | 지속 상승 |
| 채널별 CAC(시간포함) | 채널 투입비용/신규고객 | 하락 추세 |
| 회수기간(Payback) | CAC 회수에 걸리는 월수 | 2개월 이내 |

### 주간 리뷰 포맷
- 가장 효율적 채널 1개 / 비효율 채널 1개
- 전환 병목 구간 1개
- 다음 주 실험 1개(문구/오퍼/세그먼트)
