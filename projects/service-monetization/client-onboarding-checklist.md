# 고객 온보딩 체크리스트 (자료 수집 / 킥오프)

## 1) 사전 준비
- [ ] 계약/결제 완료 확인
- [ ] 담당자/의사결정자 지정
- [ ] 커뮤니케이션 채널 확정

## 2) 자료 수집
- [ ] 현재 업무 흐름도
- [ ] 사용 중 툴/계정 목록
- [ ] 반복업무 우선순위 TOP3
- [ ] 기존 데이터 샘플
- [ ] 권한/API 키 제공 가능 여부

## 3) 킥오프 미팅 안건
1. 목표 지표 합의 (시간절감/누락감소)
2. 범위/제외 범위 재확인
3. 일정/마일스톤 확정
4. 승인 절차/배포 절차 확정
5. 리스크/대응 방식 공유

## 4) 실행 중 점검
- [ ] 주간 체크인 일정 고정
- [ ] 중간 산출물 데모
- [ ] 이슈 로그 업데이트
- [ ] 변경요청(CR) 분리 관리

## 5) 납품/인수인계
- [ ] 운영 매뉴얼 전달
- [ ] 담당자 교육 1회 완료
- [ ] 롤백/장애 대응 안내
- [ ] 1주 안정화 지원

## 6) 종료 후 후속
- [ ] 전/후 성과 리포트 전달
- [ ] 개선 과제 3개 제안
- [ ] 월 구독 전환 제안
