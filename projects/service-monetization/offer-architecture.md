# 즉시수익화 오퍼 구조 설계 (자동화/코칭/세팅)

## 1) 오퍼 계층
### Free (리드 수집)
- 무료 줌 60분
- 자동화 진단 체크리스트
- 신청폼(문제 1개 제출 필수)

### Entry (저가)
- 1:1 진단콜 60~90분
- 가격: 15만~30만
- 산출물: 우선순위 자동화 로드맵 1장

### Core (핵심 매출)
- 자동화 세팅 패키지 (2~4주)
- 가격: 150만~400만
- 산출물: 자동화 플로우 3~5개 + 운영 매뉴얼

### Recurring (구독)
- 월 유지관리/개선/장애대응
- 가격: 월 29만~79만
- 산출물: 월간 리포트 + 개선안

## 2) 타깃 세그먼트
1. 학원/교육업
2. 1인 사업자
3. 소상공인/로컬 매장
4. 개인 브랜딩 운영자

## 3) 가치 제안
- "시간 절약 + 누락 감소 + 운영 안정화"를 숫자로 제시
- 개발 자체보다 운영결과(전/후 지표) 중심으로 판매

## 4) 전환 퍼널
콘텐츠 -> 신청폼 -> 무료 줌 -> 진단콜 -> Core 결제 -> 월 구독

## 5) 제안 문구 템플릿
- "2주 내 반복업무 3개 자동화, 주당 X시간 절감 목표"
- "세팅 후 운영문서까지 제공해 재현 가능하게 만듭니다"

## 6) 리스크/방어선
- 범위 외 요청은 Change Request로 분리
- 성과 보장 문구 금지
- 외부 발송 자동화는 승인 단계 기본 적용

## 7) 30일 목표
- 무료 줌 1회
- 진단콜 10건
- Core 패키지 3건 수주
- 월 구독 2건 전환
