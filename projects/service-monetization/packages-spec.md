# 상품 3종 상세 스펙 / 범위 / 산출물

## 상품 A. 스타터 진단 + 1개 자동화 세팅
- 기간: 3~5일
- 가격: 49만~99만
- 범위:
  - 현행 업무 진단
  - 자동화 대상 1개 선정
  - MVP 자동화 플로우 1개 구축
- 산출물:
  - 진단 리포트 1부
  - 자동화 플로우 문서 1부
  - 운영 가이드 1부

## 상품 B. 운영 자동화 패키지 (핵심)
- 기간: 2~4주
- 가격: 150만~400만
- 범위:
  - 자동화 플로우 3~5개
  - 데이터 정리/리포트 자동화
  - 운영자 교육 1회
- 산출물:
  - 자동화 워크플로우 세트
  - 대시보드/리포트 템플릿
  - SOP 문서

## 상품 C. 월 유지관리 구독
- 기간: 월 단위
- 가격: 29만~79만/월
- 범위:
  - 장애 대응
  - 월간 개선 1~2건
  - 운영 성과 리포트
- 산출물:
  - 월간 운영 리포트
  - 개선 이력 로그
  - 다음달 개선 제안

---

## 공통 제외 범위
- 광고 집행 대행
- 법률/세무 자문
- 외부 시스템 권한 미제공 영역

## 변경관리
- 범위 외 요청은 CR(Change Request)로 분리
- 일정/비용 영향도 사전 합의

## 품질 기준
- 재현 가능 문서 제공
- 운영자 인수인계 완료
- 승인 후 배포
