# 02. Problem Inventory

## Top 5 문제 (우선순위)
1) 문의/상담 응답 지연
   - 빈도: 매일
   - 영향: 리드 이탈, 신뢰 하락, 매출기회 손실
2) 리드 상태 추적 부재
   - 빈도: 주 3~5회 누락
   - 영향: 재접촉 실패, 제안서 미회수
3) 반복 리포트/정리 작업 과다
   - 빈도: 주간/월간 반복
   - 영향: 대표의 전략/영업 시간 잠식
4) 도구 파편화(메신저/시트/노션/메일 분산)
   - 빈도: 상시
   - 영향: 정보 누락, 책임소재 불명확
5) 자동화 시도 후 운영 정착 실패
   - 빈도: 월 단위 재발
   - 영향: “자동화 피로”, 도입 불신 증가

## 현재 대안 / 왜 실패했는가
- 대안1: 수기 체크리스트, 개인 메모
- 대안2: 단일 툴 부분 자동화(예: 알림만 자동화)
- 실패 이유:
  - 프로세스 정의 없이 도구부터 도입
  - 예외상황/책임자/운영 규칙 부재
  - 유지보수 주체가 없어 2~4주 내 붕괴

## 문제 우선순위 결론
- 1차 해결: **응답 지연 + 리드 추적 누락**
- 2차 해결: 리포트 자동화 + 운영 룰 문서화
