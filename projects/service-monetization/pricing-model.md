# 가격정책 / 할인 / 업셀 / 월구독 모델

## 1) 기본 가격 체계
### Starter (1회)
- 49~99만원
- 진단 + 자동화 1개

### Core (2~4주)
- 150~400만원
- 자동화 3~5개 + 운영 문서

### Care (월 구독)
- 29~79만원/월
- 장애대응 + 월간 개선 + 리포트

## 2) 할인 정책
- 런치 2주: 선착순 5명 15% 할인
- 패키지+구독 동시 계약: 첫달 구독 50% 할인
- 지인 추천: 추천인/피추천인 각 5% 혜택(상한 설정)

## 3) 업셀 경로
- Starter -> Core: 진단 결과 기반 제안
- Core -> Care: 운영 안정화/지속 개선 니즈 전환
- Care -> 고급코칭: 월 1회 전략 세션 추가

## 4) 결제 옵션
- 1회성: 계약금 50% + 완료금 50%
- 고가 패키지: 2~3회 분할
- 구독: 월 자동결제(해지 정책 명시)

## 5) 수익성 기준
- 최소 목표 마진율: 50%
- 시간당 최소 매출 기준 설정
- 저마진 고객은 범위 축소/템플릿화로 전환

## 6) 가격 실험
A/B 테스트:
- A안: Starter 69만
- B안: Starter 89만
지표:
- 상담->결제 전환율
- 프로젝트 완료 후 만족도
- Core 업셀 전환율

## 7) 리스크 가드레일
- 할인 남발 금지
- 무제한 수정 금지
- 범위 밖 요청은 CR 처리
