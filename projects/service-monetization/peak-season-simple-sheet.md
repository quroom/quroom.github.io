# 명절 주문 수량표 (아주 간단)

> 목적: **시간대별로 몇 개 준비하면 되는지** 바로 보기

사용 방법(3단계)
1. 주문이 들어오면 아래 표에 한 줄 추가
2. 시간대별 합계를 아래 "자동 합계" 표에 적기
3. 합계만 보고 박스 포장 수량 준비

---

## 1) 주문 입력표

| 주문자 | 품목 | 수량 | 찾으러 오는 시간 | 준비완료 |
|---|---|---:|---|---|
| 김OO | 깨송편 | 2 | 09시 | ⬜ |
| 이OO | 콩송편 | 1 | 09시 | ⬜ |
| 박OO | 깨송편 | 3 | 10시 | ⬜ |
| 최OO | 콩송편 | 2 | 10시 | ⬜ |
| 정OO | 깨송편 | 1 | 11시 | ⬜ |

> "준비완료" 체크는 포장 끝났을 때만 체크

---

## 2) 시간대별 합계(핵심)

| 시간대 | 깨송편(개) | 콩송편(개) | 총 수량(개) | 미리 포장 수량(개) |
|---|---:|---:|---:|---:|
| 09시 | 2 | 1 | 3 | 3 |
| 10시 | 3 | 2 | 5 | 5 |
| 11시 | 1 | 0 | 1 | 1 |
| 12시 | 0 | 0 | 0 | 0 |
| 13시 | 0 | 0 | 0 | 0 |
| 14시 | 0 | 0 | 0 | 0 |
| 15시 | 0 | 0 | 0 | 0 |
| 16시 | 0 | 0 | 0 | 0 |
| 17시 | 0 | 0 | 0 | 0 |

규칙(간단):
- "미리 포장 수량" = "총 수량"과 같게 시작
- 당일 현장 판매가 많으면 +1~2개 여유분 추가

---

## 3) 절대 규칙(중요)

1. **예약분 먼저 빼두기**
2. 예약분은 현장 판매에 섞지 않기
3. 찾으러 온 고객은 이름+시간 확인 후 전달
4. 전달하면 주문표 "준비완료" 체크

---
