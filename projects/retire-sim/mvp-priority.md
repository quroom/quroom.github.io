# MVP 기능 우선순위 (MoSCoW)

## Must (v1 필수)
1. 기본 입력 폼 (자산/적립/지출/은퇴나이/물가/수익률)
2. 시뮬레이션 엔진 실행 (기준 시나리오)
3. 결과 카드 (달성확률/부족분/조정안)
4. 보수/중립/낙관 3시나리오 비교
5. 저장/재실행 (입력값 재사용)
6. 가정/주의문구 표시 (투자권유 아님)

## Should (v1.1 권장)
1. 몬테카를로 1,000회 옵션
2. 은퇴 후 인출률 조절 슬라이더
3. 연금/기타현금흐름 상세 입력
4. 결과 PDF 리포트 내보내기
5. 입력 유효성 경고/가이드

## Could (v1.2 이후)
1. 자산군별 비중/리밸런싱 반영
2. 세금 상세모형(계좌유형별)
3. 부부합산 시뮬레이션
4. 목표 이벤트(주택/교육비) 반영
5. 주기적 리마인드 알림

## Won’t (현재 제외)
1. 종목 추천/타이밍 신호
2. 단기 매매 전략 기능
3. 고빈도 트레이딩/파생상품 모드
4. 자문업 대체 문구/기능

---

## 우선순위 판단 기준
- 사용자 가치(결정 피로 감소/계획 유지 도움)
- 구현 난이도 대비 효과
- 규제/오해 리스크
- 반복 사용성(재방문 유도)

## 릴리즈 게이트
- Must 100% + Should 2개 이상 충족 시 v1 공개
