# 노후준비 자산관리 시뮬레이션 서비스 PRD v1

## 1) 제품 개요
- 제품명(가칭): RetireSim
- 목적: 지수투자 관점에서 은퇴자금 준비 가능성을 수치로 확인하고, 과도한 판단 없이 장기 계획을 유지하도록 돕는다.
- 핵심 가치: 예측이 아니라 **구조 점검** (적립/인출/물가/세금/변동성 내성)

## 2) 대상 사용자
1. 30~50대 직장인: 은퇴자금 목표는 있지만 현재 계획의 현실성을 모름
2. 장기 지수투자자: 시장 소음보다 계획 유지 가능성(성공확률/부족분)을 알고 싶음
3. 금융 초보: 매수 타이밍보다 월 적립/지출/목표 은퇴연령을 기반으로 간단히 점검하고 싶음

## 3) 문제 정의
- 현재 자산/적립금으로 목표 은퇴생활비를 달성 가능한지 직관적으로 확인하기 어려움
- 하락장/변동성 상황에서 계획을 자주 바꾸게 됨
- 세금/물가/인출률을 고려한 장기 시뮬레이션 도구가 부족함

## 4) 제품 목표 (v1)
- 10분 내 개인 은퇴 시뮬레이션 1회 완료
- 결과: 목표 달성 확률, 부족자금, 권장 조정안(적립액/은퇴시점/지출)
- 사용자 행동 목표: "매매 판단" 대신 "계획 점검" 중심으로 전환

## 5) 비목표 (v1)
- 종목 추천/매수 타이밍 제시
- 단기 수익률 예측
- 세무/법률 자문 대체

## 6) 핵심 기능 (v1)
1. 입력
   - 나이, 은퇴 희망 나이
   - 현재 투자자산
   - 월 적립금
   - 목표 은퇴 후 월지출(현재가치)
   - 기대 인플레이션, 기대 실질수익률(보수적 기본값 제공)
   - 연금/기타 현금흐름
2. 시뮬레이션
   - 기준 시나리오
   - 보수/중립/낙관 시나리오
   - 간단 몬테카를로(옵션)
3. 결과
   - 목표 달성 가능성(%)
   - 은퇴 시점 예상 자산 범위
   - 부족자금 및 보정 옵션 3개
4. 리포트
   - 저장/재실행/비교
   - 월간 점검 리마인드

## 7) 핵심 지표 (KPI)
- 시뮬레이션 완료율
- 결과 저장률
- 7일 재방문율
- 재시뮬레이션율
- 계획 조정 실행률(적립액 변경/은퇴시점 조정)

## 8) 사용자 플로우
1. 온보딩(목적/주의사항)
2. 기본값 자동 채우기
3. 입력 검증
4. 시뮬레이션 실행
5. 결과 카드 + 조정안 제시
6. 리포트 저장/공유

## 9) 리스크/가드레일
- 투자권유로 오해될 수 있는 표현 금지
- 결과에 "불확실성"과 "가정" 명시
- 사용자가 과도하게 빈번한 입력 변경을 하지 않도록 가이드

## 10) 릴리즈 계획
- v1 (2~4주): 입력/시뮬레이션/결과 카드
- v1.1: 시나리오 비교 + 리포트 히스토리
- v1.2: 연금/세금 상세 반영, API 확장
