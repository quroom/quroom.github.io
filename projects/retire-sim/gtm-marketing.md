# GTM / 마케팅 플랜 (콘텐츠/채널/지표)

## 1) 포지셔닝
- 메시지: "투자판단 도구가 아니라 은퇴계획 점검 도구"
- 타깃: 30~50대 장기 지수투자자, 계획 불확실성 해소 니즈 보유층

## 2) 채널 전략
1. Threads
- 목적: 도달/관심 유입
- 형식: 짧은 인사이트 + 체크리스트 + CTA

2. LinkedIn
- 목적: 신뢰 확보/전문성 전달
- 형식: 케이스 스토리, 데이터 기반 포스트

3. 네이버카페
- 목적: 전환/관계 유지
- 형식: 장문 가이드, Q&A, 모집 공지

## 3) 콘텐츠 기둥
- 기둥 A: 은퇴 준비 현실 점검
- 기둥 B: 지수투자 장기계획 유지 전략
- 기둥 C: 시뮬레이션 해석법(오해 방지)
- 기둥 D: 실제 사례/전후 변화

## 4) 4주 실행 캘린더
- 주 1: 문제 인식(왜 계획 점검이 필요한가)
- 주 2: 방법 제시(시뮬레이션 사용법)
- 주 3: 사례 공개(가명 케이스)
- 주 4: 오퍼 전환(진단/코칭/패키지)

발행 빈도:
- Threads: 주 5회
- LinkedIn: 주 2회
- 카페: 주 2회 + 주간 Q&A 1회

## 5) 전환 퍼널
콘텐츠 -> 신청폼 -> 진단콜 -> 유료 전환 -> 후기/리퍼럴

## 6) KPI
- 상단: 도달수, 저장/공유율
- 중단: 폼 전환율, 상담 예약률
- 하단: 결제 전환율, 객단가, 재구매/유지율

## 7) 실험 항목
- CTA 문구 A/B
- 랜딩 섹션 순서 A/B
- 무료 진단 vs 저가 진단 전환 비교

## 8) 리스크/완화
- 과장 표현 금지
- 투자권유 오해 방지 문구 고정
- 결과 해석 가이드 동시 제공
