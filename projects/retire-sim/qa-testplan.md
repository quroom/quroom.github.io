# QA 테스트플랜 + 수치 검증 케이스

## 1) 테스트 범위
- 입력 유효성
- 계산 정확성(적립/인출/물가/세금)
- 시나리오 비교 일관성
- 결과 표시/내보내기

## 2) 기능 테스트 케이스
1. 필수값 누락 시 에러
2. 음수 입력 차단
3. 은퇴나이 < 현재나이 차단
4. 극단값 경고 노출
5. 시나리오 전환 시 결과 갱신
6. 결과 저장/불러오기 정상

## 3) 수치 검증 케이스 (Golden Set)
### Case A: 단순 적립
- 입력: 초기자산 1억, 월100만, 연수익 5%, 10년
- 기대: 연복리 계산과 오차 < 0.5%

### Case B: 인출 포함
- 입력: 은퇴자산 8억, 연지출 3천, 연금 1천, 수익 4%
- 기대: 20년 후 잔액 범위 확인

### Case C: 물가 반영
- 입력: 지출 300만, 물가 3%, 15년
- 기대: 지출 증가곡선 검증

### Case D: 세금 드래그
- 입력: 명목수익 6%, taxDrag 10%
- 기대: 실효수익 5.4% 적용 확인

### Case E: 시나리오 비교
- 입력 동일, 수익률만 보수/중립/낙관 변경
- 기대: 결과 순서 보수 < 중립 < 낙관

## 4) 비기능 테스트
- 응답시간: 단일 시뮬 < 2초(기준 모드)
- 안정성: 100회 반복 시 오류율 < 1%
- 재현성: 동일 seed 동일 결과

## 5) 회귀 테스트 정책
- 엔진 변경 시 Golden Set 전체 재실행
- 결과 차이 발생 시 릴리즈노트에 명시

## 6) 릴리즈 게이트
- P0 버그 0건
- 수치 검증 케이스 100% 통과
- 고지/면책 문구 누락 0건
