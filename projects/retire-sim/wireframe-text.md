# 화면설계 와이어프레임 (텍스트 버전)

## 화면 1) 랜딩/온보딩
- 헤더: 서비스명, "투자권유 아님" 안내
- 본문: 핵심 가치 3줄
- CTA: [시뮬레이션 시작]
- 보조: 샘플 리포트 보기

## 화면 2) 입력 폼
- 섹션 A: 기본정보 (나이/은퇴나이/기대수명)
- 섹션 B: 자산/적립 (현재자산/월적립/적립증가율)
- 섹션 C: 은퇴지출/연금 (월지출/연금유입)
- 섹션 D: 가정 (수익률/물가/세금)
- 하단: [기본값 복원] [시뮬레이션 실행]

## 화면 3) 결과 요약
- 카드 1: 목표 달성 확률
- 카드 2: 은퇴 시점 예상자산(중앙값)
- 카드 3: 부족자금
- 카드 4: 고갈확률

## 화면 4) 시나리오 비교
- 탭: 보수 / 중립 / 낙관 / 스트레스
- 표: 달성확률, 부족분, 권장 조정안
- 버튼: [내 계획으로 저장]

## 화면 5) 조정 시뮬레이터
- 슬라이더: 월적립금, 은퇴시점, 지출
- 실시간 업데이트: 결과 카드 값 변경
- CTA: [이 설정 저장]

## 화면 6) 리포트/히스토리
- 최근 실행 목록
- 각 실행 비교 버튼
- PDF 내보내기

## 화면 7) 설정/고지
- 개인정보 최소수집 안내
- 모델 가정/한계
- 투자권유 아님 고지

---

## UX 원칙
- 숫자 입력은 최소화, 기본값 제공
- 결과는 "행동"보다 "계획 점검" 중심
- 경고 문구는 불안 유발 없이 중립적으로 표기
